# 🎮 DigitStore - Цифровой магазин игр

Современный веб-сайт для покупки и продажи цифровых товаров, включая игры, DLC и игровые аккаунты.

## 🌟 Возможности

- 🎯 **Каталог товаров** - Полный каталог игр, DLC и аккаунтов
- 🔍 **Поиск и фильтрация** - Поиск по названию, фильтрация по категориям и цене
- 🛒 **Корзина** - Добавление товаров в корзину с сохранением в LocalStorage
- 📱 **Адаптивный дизайн** - Отлично работает на мобильных устройствах
- 💳 **Система оплаты** - Готовность к интеграции платежей
- ⚡ **Быстрая загрузка** - Оптимизированная производительность

## 🛠️ Технологии

- **HTML5** - Семантическая разметка
- **CSS3** - Современные стили с градиентами и анимациями
- **JavaScript (ES6+)** - Функциональность без зависимостей
- **LocalStorage** - Сохранение корзины в браузере

## 💡 Как использовать

1. **Поиск товаров** - Используйте поисковую строку
2. **Фильтрация** - Выберите категорию или диапазон цен
3. **Добавление в корзину** - Нажмите кнопку "В корзину"
4. **Просмотр корзины** - Нажмите на иконку корзины в верхнем правом углу
5. **Оформление** - Нажмите "Оформить покупку"
